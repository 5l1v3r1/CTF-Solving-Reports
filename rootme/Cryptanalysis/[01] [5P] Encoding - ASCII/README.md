水题，题目就是提示，ASCII 解码。

打开挑战页面只看见一串十六进制文本。

打开 Burp Suite -> Decoder ，使用 ASCII hex 解码器解码，还原 ASCII 字符。

冒号后面的就是密码，完成挑战。
